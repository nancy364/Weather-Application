🚀 Weather App Project

I’m excited to share one of my recent projects — a **Weather Application** built using **HTML, CSS, and JavaScript**.

This application allows users to search for any city and instantly get real-time weather information. It fetches live data from a weather API and displays important details such as **temperature, weather conditions, and country information** in a clean and user-friendly interface.

✨ **Key Features:**
• Search weather by city name
• Real-time weather data using API integration
• Dynamic UI updates based on search results
• Error handling for invalid city searches
• Responsive and modern UI design

Through this project, I strengthened my understanding of **API integration, DOM manipulation, asynchronous JavaScript, and responsive UI design**.

Looking forward to improving this project further and adding more advanced features.

💻 **Tech Stack:** HTML | CSS | JavaScript | Weather API
